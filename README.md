# Book Scraper

## Description

A simple Python project that collects book data from a website using web scraping.

## Features

* Extracts book titles
* Extracts prices
* Extracts ratings
* Saves the data in a CSV file

## Tools Used

* Python
* Requests
* BeautifulSoup
* Pandas

## How to Run

1. Install the required libraries
   pip install requests beautifulsoup4 pandas

2. Run the script
   python scraper.py

## Output

The scraped data will be saved in a file called *books.csv*.
